Sass 與 WordPress

使用SCSS修改樣式

首先是用NPM安裝node-sass，再撰寫腳本指定覆蓋原本的style.css

../package.json

將原本的style.css改名為 _styleWP.scss，利用main.scss匯入後，再利用css權重去加入客製化樣式，如此一來就可以在舊有樣式上進行修改。

./scss/main.scss
